Introduction

Memory works in the brain much like the rest of the brain works. In a very malleable way. Memory isn't the same forever, it's constantly changing, everytime we remember it.

When it comes to memory, most people think of it in terms of a filing system, where a memory is archived in a single place, remaining like a book on a shelf, until desired.
